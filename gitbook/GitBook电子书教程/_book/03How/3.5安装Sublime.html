第五节 安装Sublime

Sublime text 3 汉化 破解版 分享

百度云下载 密码：7fzl

个人百度网盘中也有详细配置和已经配置好的安装包。不再细谈。

1.按住 Ctrl + shift + p，输入 package control

2.点击 选项 package control ：Install Package，等待插件安装界面的响应

3.输入想要安装的插件名，如 HTML Beautiful，JSONlint等

4.点击匹配的查找结果，等待安装

如遇到更新失败，网络延时，网络中断情况。

1、多试几次

2、修改hosts文件。
