# 摘言拾句

```
天行健，君子以自强不息。
```

> 一等人不用教，二等人用言教，三等人用棍教
>
> 《周易》64卦，乾卦第一，当头就说：天行健，君子以自强不息。
>
> 爹好，娘好，不如自强好。
>
> 男儿当自强。自强不息。
>
> ``` 
> God helps those who help themselves.上帝帮助自助之人。
> ```

```
孙子曰："昔之善战者，先为不可胜，以待敌之可胜。不可胜在己，可胜在敌。”
```

> 善于指挥作战的人，先要做到不会被敌人所战胜，然后待机战胜敌人。
